Juliaで動かす金融計算-先渡のシミュレーション編¶

原資産となる株価のシミュレーションができたので, それを使ったデリバティブにも挑戦してみましょう.
ここでは, シンプル（？）なデリバティブとして, 先渡（フォワード）を題材に扱います.

先渡（フォワード）って？¶

先渡（フォワード）取引とは, あらかじめ決められた日に, 決められた価格で原資産（今回は株にしましょう）を売買する取引です. 事前にいつ, いくらで取引するのかを最初に決めておくのが特徴ですね.
似たような取引に, 先物（フューチャー）取引があります. 両者の違いはいろいろあるのですが, ここでは置いておきましょう.
もう一つ, 先渡取引で大事な点があります. それは, 将来の売買が義務である点です.
これを覚えておくと, のちに扱うオプションの話が分かりやすいと思います.

先渡の価値計算¶

それでは早速、先渡にかかわる計算をやってみましょう.
先渡の計算として, ここでは自分が契約した先渡（買い）の価値がどのように推移していくのかを, グラフで表してみましょう.
そのためには, 時々刻々変化する株価に対して, 自分が契約した先渡（買い）の価値を求める必要があります.

先渡の価値計算方法¶

その先渡の価値をどうやって計算するのかについて, 説明しておきましょう.
先渡（買い）の価値ですが, 以下のように計算されます.
$$ 先渡の価値 = 現時点での株価 - ディスカウントファクター \times 先渡価格 $$ 先渡価格というのは, 契約したときに決めた価格で, ある時点（=満期）での取引価格を表します.
ディスカウントファクターというのは, 将来時点での価値を, 現在時点への価値へと変換する掛け目です.

将来の価値と現在の価値¶

金融の世界では, たとえ同じ100円であっても時点が違えば価値が違います.
あまりピンとこないかもしれませんので, 例をとって説明します.
今100円持っているとしましょう. これを銀行に預けた際の預金金利が10%/年だったとします（超高金利！）. こうすると今の100円は1年後に110円になります. つまり, 今の100円は1年後の110円と同じなのです. 逆に言えば, 1年後の110円は現在の価値に換算すると, 100円分の価値しかありません.
このように, 時点が違えば価値が異なるのです. この, 将来のものを現時点での価値に換算することを割り引くと言います.

先渡の価値の計算まとめ¶

では, これを踏まえて改めて先渡取引を考えてみましょう. 先渡取引では, 将来のある時点で原資産（今回は株式でしたね）を売買する取引でした. つまり先渡（買い）の価値は, 満期での株式と先渡価格の差になります.
しかし, 満期での株式の価格を求めることはちょっと難しいです. 満期での株式の価格が完璧にわかるなら今頃億万長者ですからね. そこで, すべてを現時点での価値に直して考えることにするのです.
現時点で考えると, 株式の価値は当然現在の価格です. 先渡価格を現在時点の価値に割り引くためには, ディスカウントファクターを掛ければよいです.

今まで説明したことを図にまとめてみました. 大事なことは, 以下の2つです.

  1. すべてを現時点で考える
  2. 割り引く際にはディスカウントファクターをかける

これさえ覚えておけば何とかなるでしょう.

ディスカウントファクターの計算関数¶

次に, ディスカウントファクターの計算をする関数を作成してしまいましょう.
ディスカウントファクターの計算方法がなぜ以下でいいのかは, 余裕があれば考えてみてください.
基本的に, ここは「こんなもんか」で済ませてくれればと思います.

#時刻Tの価値を時刻tの価値にするディスカウントファクターの計算をする関数
#rは金利
function CalcDiscountFactor(r, T, t)
    return exp(-r * (T - t))
end

先渡の価値のシミュレーション¶

では, 株価を作りながら, 先渡の価値を計算して, グラフに表示してみましょう.
処理の流れとしては, 以下のようです.

  1. 定数の設定
  2. 時間の生成
  3. 正規分布を作成
  4. 初期設定
  5. 株価の生成
  6. ディスカウントファクターを計算
  7. 先渡の価値計算
  8. プロット

手順が長く感じるかもしれませんが, 実は途中はほぼ株価のシミュレーションと同じです.
では, コードを書いてみましょう.

In [6]:
using Distributions
using StatsPlots

#先渡の価値を計算する関数
function CalcForwardValue(S, df, K)
    return S - df * K
end

#時刻Tの価値を時刻tの価値にするディスカウントファクターの計算をする関数
#rは金利
function CalcDiscountFactor(r, T, t)
    return exp(-r * (T - t))
end

function SimulateForwardValue()
    #1.定数の設定
    T = 1
    Δt = 0.01
    μ = 0.5
    σ = 0.2
    S0 = 100.0 #初期株価
    K = 120
    r = 0.01 #金利

    #2.時間の生成
    t = 0:Δt:T

    #3.正規分布を生成
    d = Normal(0, Δt^0.5)

    #4.初期設定
    S = [S0]
    df = CalcDiscountFactor(r, T, t[1])
    V0 = CalcForwardValue(S[1], df, K)
    V = [V0]
    
    #5.時間ごとに株価を生成
    for t_i in t[2:end]
        constantDiff = μ * Δt * S[end] #確定的な変動
        w = rand(d) #乱数生成
        randomDiff = σ * w * S[end] #ランダムな変動
        nextS = S[end] + constantDiff  + randomDiff #次の株価を計算
        push!(S, nextS) #次の株価を配列に格納

        #6.ディスカウントファクターの計算
        df = CalcDiscountFactor(r, T, t_i)

        #7.先渡の価値を計算する
        nextV = CalcForwardValue(nextS, df, K)
        push!(V, nextV)
    end

    #8.プロット
    p1 = plot(t, S, label = "StockPrice") #株価をプロット
    plot!(p1, t, fill(K, length(t)), label = "K") #株価のプロットにKもプロット. fill(K, n)で長さがnで要素がすべてKの配列が作れる

    p2 = plot(t, V, label = "ForwardValue")#先渡の価値をプロット

    plot(p1, p2)
end

SimulateForwardValue()

ほとんど, 株価と同じような動き方をしていることがわかるでしょう.
